TV Guide displays the user's list of preferred channels in the Channels pane of the main view:

Note: channel numbers are only available when using the Australian TV programme data from the OzTivo guide service.

If some channels are not appropriate in your region or you do not watch them regularly, you can refine the list using the Edit Channels option on the Tools menu.

Next, select the specific date you wish to view:

Toolbar buttons can also be used to select the previous day, next day, previous week, next week, and today:

Once a channel and date have been selected, the programmes will be fetched from the guide service and displayed in the main view:

If you hover the mouse over a specific programme, then further details about the programme will be displayed:

Options on the View menu can be used to scroll to a specific time of day:

All Channels from the Channels pane will show the programmes for the current day for all channels in columns:

Because All Channels may be a lot of channels, you can create channel groups to separate the channel list into drama, movies, comedy, and so on.
