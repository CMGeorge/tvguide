TV Guide can import and export your bookmarks in an XML-based format, which allows you to share them with others or copy them between multiple machines that you own that all run TV Guide.

Export on the Bookmarks menu will export all of the bookmarks and ticks to a file with the tvb file extension.

Import on the Bookmarks menu will import the contents of a tvb file and merge the new bookmarks and ticks with your current list. If you already have a bookmark that is listed in the file, then TV Guide will skip that entry rather than create a duplicate.

Note: bookmarks are specific to the guide service that you are using, so you can only share them with other users that have the same guide service. If you are using the OzTivo service, and you import bookmarks from another region, then the bookmarks will be automatically converted to your region.
