The main programme view can be filtered with a search string from the toolbar:

By default, when you type a string into this box, it will restrict the view to the programmes that contain the string within their titles. You can search on other criteria using the search options pull-down menu:

Only one of Title, Episode Name, Description, Credits, and Categories can be used to conduct the search.

Select Credit and Select Category can be used to select a specific item from the programme details that have been previously loaded into memory. Note: if no programmes have been loaded yet, then no credits or categories will be available for selection.
