The Edit Channels option on the Tools menu can be used to refine the list of channels that are displayed by TV Guide:

and can be used to move channels from the visible list to the hidden list and in reverse. and can be used to hide or show all channels. Finally, and can be used to add and remove channel-specific logo images.

Note: region selection is only available when using the Australian TV programme data from the OzTivo guide service.
