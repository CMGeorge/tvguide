TV Guide can remember programmes from week to week to highlight them whenever they appear. First, click on a programme you are interested in bookmarking:

Then select Add Bookmark from the Bookmarks menu, which will display the information about the programme:

Season can be left unchecked to match the show in any season. Seasons can be specified by range with 3-6 or 3+, or separated by commas as 1,3,7. Similarily, Year can be checked to specify a year range.

Press OK to save the bookmark and whenever a programme appears in the programme list that matches the bookmark, it will be displayed in the selected color:

If you use Add Bookmark on a programme that is already bookmarked, then you will be asked if you wish to edit the existing bookmark's details or add a new bookmark.

The Day drop down list provides a selection of days for the bookmark: Any day, Monday through Sunday, Monday to Friday, or Saturday and Sunday. If you need some other combination of days, then press the Other button and select the days manually:

7 Day Bookmark Outlook on the Bookmarks menu will show the bookmarked programmes for the next 7 days on the current channel, starting at the selected date in the calendar:

It is also possible to set a bookmark on a programme that isn't currently shown in the programme view. This can allow you to watch out for an upcoming programme that hasn't started yet. It will be automatically highlighted in the regular view and the 7 day outlook when the programme starts showing.

If a bookmark matches on the programme name, but not on the channel, day, or time, then the title will be shown in non-bold. In the following example, the bookmark has been set on the 7:30 PM timeslot:

Combined with the 7 Day Bookmark Outlook feature, partial matches can make it easier to locate a show after it has moved to a different channel, day, or time.

If the bookmark is marked as On Air and the programme is no longer shown in its bookmarked timeslot, then the programme will be shown with a strike-through of the bookmark's show title:

If TV Guide has seen the same show in a different timeslot or on a different channel, then it may list those on the popup tooltip so you can update your bookmarks:

You can look for partial matches to see if the show has moved to another channel or time. If it has moved, then update the bookmark to reflect the new details. If however the show is no longer on, but may return at a later date, then change the bookmark to Off Air. When the show returns, it will be shown as follows:

Organize Bookmarks on the Bookmarks menu can be used to edit, delete, and create new bookmarks.
