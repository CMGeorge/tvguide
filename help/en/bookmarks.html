TV Guide can remember programmes from week to week to highlight them whenever they appear. First, click on a programme you are interested in bookmarking:

Then select Add Bookmark from the Bookmarks menu, which will display the information about the programme:

Season can be left unchecked to match the show in any season. Seasons can be specified by range with 3-6 or 3+, or separated by commas as 1,3,7.

Press OK to save the bookmark and whenever a programme appears in the programme list that matches the bookmark, it will be displayed in the selected color:

If you use Add Bookmark on a programme that is already bookmarked, then you will be asked if you wish to edit the existing bookmark's details or add a new bookmark.

The Day drop down list provides a selection of days for the bookmark: Any day, Monday through Sunday, Monday to Friday, or Saturday and Sunday. If you need some other combination of days, then press the Other button and select the days manually:

7 Day Bookmark Outlook on the Bookmarks menu will show the bookmarked programmes for the next 7 days on the current channel, starting at the selected date in the calendar:

It is also possible to set a bookmark on a programme that isn't currently shown in the programme view. This can allow you to watch out for an upcoming programme that hasn't started yet. It will be automatically highlighted in the regular view and the 7 day outlook when the programme starts showing.

If a bookmark matches on the programme name, but not on the channel, day, or time, then it will be shown in a slightly darker version of the color. In the following example, the bookmark has been set on the 7:30 PM timeslot:

This feature can be enabled or disabled using Show Partial Matches on the Bookmarks menu. Combined with the 7 Day Bookmark Outlook feature, partial matches can make it easier to locate a show after it has moved to a different channel, day, or time.

Show Failed Matches on the Bookmarks menu will show bookmarks that fail to match the programme exactly with a strike-through of the bookmark's show title. This can be used to notice when a show is no longer playing or it may have moved to another channel, day, or time:

Organize Bookmarks on the Bookmarks menu can be used to edit, delete, and create new bookmarks. Individual bookmarks can be temporarily disabled for matching by unchecking the bookmark in the list.
