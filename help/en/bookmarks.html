TV Guide can remember programmes from week to week to highlight them whenever they appear. First, click on a programme you are interested in bookmarking:

Then select Add Bookmark from the Tools menu, which will display the information about the programme:

Press OK to save the bookmark and whenever a programme appears in the programme list that matches the bookmark, it will be displayed in the selected color:

7 Day Bookmark Outlook on the View menu will show the bookmarked programmes for the next 7 days on the current channel, starting at the selected date in the calendar:

It is also possible to set a bookmark on a programme that isn't currently shown in the programme view. This can allow you to watch out for an upcoming programme that hasn't started yet. It will be automatically highlighted in the regular view and the 7 day outlook when the programme starts showing.

Organize Bookmarks on the Tools menu can be used to edit, delete, and create new bookmarks.
